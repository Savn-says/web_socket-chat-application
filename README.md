# 💬 Real-Time Chat Application (WebSocket)

> A full-stack chat system enabling real-time communication between multiple clients using WebSockets.

---

## 🧠 Overview

This project demonstrates real-time bidirectional communication using WebSockets.
Messages are instantly broadcasted across all connected clients without page refresh.

---

## 🏗 Architecture

Client (React) ⇄ WebSocket (STOMP) ⇄ Spring Boot Server

---

## 🛠 Tech Stack

* **Frontend:** React (Vite), CSS
* **Backend:** Spring Boot
* **Protocol:** WebSocket (STOMP)
* **Language:** Java

---

## ⚡ Features

* Real-time messaging
* Multi-client synchronization
* Persistent WebSocket connection
* Clean full-stack separation

---

## 🚀 How to Run

### Backend

```bash
cd Demo_WebSocket
./mvnw spring-boot:run
```

### Frontend

```bash
cd frontend/websocket
npm install
npm run dev
```
